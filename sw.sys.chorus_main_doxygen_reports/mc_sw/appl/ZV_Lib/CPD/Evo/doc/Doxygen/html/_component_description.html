Component Description

General

The SW component CPD implements the application logic for the detection of a ChildPresenseDetection state.
The defined user warning outputs are calculated and output by means of logical signals.
The SW component CPD processes information about the door states of the rear doors and information about the terminal control.
The CPD SW component processes information about the personalization settings of the ChildPresenseDetection function supplied by a personalization software provided by the integrator.

CPD currently integrates library functions(init_cpd(), cyclic_cpd()) and connections with Components that are established through RTE.
Note: Components are those components that CPD interacts with, for more details, see VW_ICAS1.Chorus.All_Cores.SoftwareDecomposition.ApplicationLayer.ZV_Lib.SW Architecture.CPD.CPD Static.CPD Context
