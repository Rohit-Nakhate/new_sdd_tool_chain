Internal Component Design

Internal Component Design

The component is splitted into the following SW units:

  • KFTX_Task.c - This module will run the KFTX functionality, monitor some errors status and sets customization settings based on BAP usage
  • KFTX_Task.h - Header file for KFTX_Task.c
  • KFTX_DataTypes.h - This headerfile provides the datatypes used to provide the S/R Autosar functions for coding,adaption etc.

Requirements Allocation

All the software requirements assigned to KFTX are satisfied by KFTX_Task.c . KFTX module has no non functional requirements .
