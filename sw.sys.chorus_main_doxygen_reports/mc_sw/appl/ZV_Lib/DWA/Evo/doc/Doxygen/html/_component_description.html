Component Description

General

The SW-component DWA implements the application logic of the anti-theft warning system. It determines and controls the state of the anti-theft warning system, evaluates the alarm sources and triggers an DWA-alarm according to the alarm sources.
Currently, DWA:
- integrates the cyclic_dwa() function from the DWA library
- integrates the DWAX functionality by calling the init,cyclic and stop functions (DWAX is an extension of DWA)
- sets certain errors for DWA Sensor and DWA Sirene (DTCs)
- establishes Sounder LIN communication
- integrates connections with Components that are established through RTE.

Note: Components are those components that DWA interacts with, for more details, see the DWA architecture.
