Component Description

General

The SW-component ZV implements the application logic of central door locking.
The SW component processes locking and unlocking requests, determines the target state and the actual state of the central door locking and provides data elements to control the central door locking latches and optic and acoustic acknowledgement and feedbacks.

ZV component integrates the ZV 3rd party library delivered by VW, with the other SWCs (for more details check the ZV Dependencies diagram present in SWA) established through RTE.
