Internal Component Design

Internal Component Design

The component is splitted into the follwing SW units:

  • KFT_Task.c - Code is needed for extending CAN frames duration, it is called by scheduler which will init, start and stop the operation of KFT
  • KFT_Task.h - Header file for KFT_Task.c
  • KFT_Stubs.c - Lib with stubs is needed for testing KFT related interfaces
  • KFT_Stubs.h - Header file for KFT_Stubs.c
  • KFT_NvRam.h - Header file is needed for connecting memory and lib
  • KFT_ExtraDefinitions.h - This headerfile controls the overwriting of generated S/R autosar_functions
  • KFT_MemMap.h
