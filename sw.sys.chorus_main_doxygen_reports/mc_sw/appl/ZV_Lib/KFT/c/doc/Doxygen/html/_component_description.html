Component Description

General

The KFT SW component implements the application logic of the enabling operations and comfort functions for the electric window lifters, for the electric tilt/slide sunroof (SAD) and for the electrically drivable folding top. It controls the enabling of the electric window lifters, of the SAD and of the folding top and it issues commands for the opening and closing of the window lifters and of the SAD, based on the control information of the radio remote control, the key cylinder, the window lifter button of the driver door as well as the Kessy-control sections.
