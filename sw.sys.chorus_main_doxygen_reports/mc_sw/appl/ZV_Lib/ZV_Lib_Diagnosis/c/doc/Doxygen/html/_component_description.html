Component Description

General

This module implements ZV_Lib_diagnosis which containes four components: Adaptation, DiagControl, RoutineControl and MeasurementValues.

Adaptation component was created to handle diagnostic requests for certain DIDs of the ZV_Lib composition and it is used to perform the read(22 service) - write(2E service) operations at system level.
DiagControl component was created in order to handle the preconditions of starting a routine and it is used to perform the requests from DCM to start/stop/requestResults for a routine using service 31 (RoutineControl).
RoutineControl component was created to handle the abort routines.
MeasurementValues main purpose is to read data using 22 service, get the data from different application and BSW component and give it to the tester.
