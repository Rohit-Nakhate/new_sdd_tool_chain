Document Purpose and Scope

Document Purpose

This document contains the Software Detailed Design documentation for BAP base libraries used by BAPHandler SWC's in project ICAS1.

The BAP base library BAPHandler_PSO_ImExport handle personalisation functionality for BAPHandler SWC's.

The BAP base library BAPHandler_UtilityLib includes universal, common functions which can be used by any SWC if needed.

Document Scope

The main purpose of this document is to describe the software detailed design of the BAP base library "BAPHandler_PSO_ImExport" as defined
in the SWA along with all allocated SWRs valid for Active Participant functionality.

Additionally it describes the software detailed design of the "BAPHandler_UtilityLib" unit.
However in this case description is limited only to Doxygen description of the defines, enums, types and functions with uml diagrams collected in this unit.
There is no specific SWA nor SWR's defined for this library. No static or dynamic bahavior was defined for this library.

Design elements for requirements only partially allocated to this component will only cover that part.
