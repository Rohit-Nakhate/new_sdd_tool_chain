Resource Consumption Objectives

The resource consumption cannot be explicitly defined for BAPHandler_PSO_ImExport and BAPHandler_UtilityLib libraries.
Potential influence on specific resource consumption should be considered from the perspective of specific SWC which is using functions from these libraries.
The library BAPHandler_PSO_ImExport serve the Active Participant functionality and handle PSO_ImExport functionality, which influences in recognized manner
the resource consumption, but should be considered as a part of resource consumption of specific SWC which is using this library (as was reflected in SWA in
relevant architecture decision).

The library BAPHandler_UtilityLib collects simple helper funtions which has minimal impact on performance, memory consumption or CPU Load and also
should be considered as a part of resource consumption of specific SWC which is using this library.

Performance:
The performance objectives were not defined for these libraries and are not mentioned in the SWA.

Memory Consumption:
The memory consumption objectives were not defined for these libraries and are not mentioned in the SWA.

CPU Load:
The CPU load objectives were not defined for these libraries and are not mentioned in the SWA.
