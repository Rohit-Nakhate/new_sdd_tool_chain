Interrupts

Libraries do not use interrupts.


Events

Libraries do not use events.


Operating Modes

Libraries do not use operating modes.


Power Modes

Libraries do not use power modes.


Exception and Error Handling

No DTC or DEM events. Error handling for wrong input values or parameters is prepared and can be found in detailed design for specific functions.
In specific situations RTE interface for informing about errors is used.


Synchronization

Libraries do not use multi-threading, so no synchronization is used.


Operations

Libraries do not perform operations other than showed in Dynamic Behavior section and represented in detailed design for specific functions.


Macros

Libraries have only one macro defined:
SWATT_BYPASS(a,b)
which is connected to testing, and widely used in many other components.


Algorithms

Algorithms used in libraries are showed in Dynamic Behavior section.
