enum BAPHandlerState_e

The enum to define the state of BAPHandler activity

Enumerator
stateBH_Start 

initial state used for BAPHandler startup to get latest adaptation and codings

stateBH_AdaptAndCoding_ready 

BAPHandler state set when adaptations and codings are ready to be used to configure BAP communication.

stateBH_BAPprotocolAndSystem_ready 

BAPHandler state used to indicate end of configuration of BAP protocol and sysytem functionalities - not used.

stateBH_check_BAP_state 

BAPHandler state used to establish BAP communication.

stateBH_BAP_postpone 

BAPHandler state used to postpone handling of BAP communicaton of 5 cycles.

stateBH_BAPfunctions_ready 

most common state of BAPHandler, used to cyclic handling of BAP functionality

stateBH_BAPHandler_ready 

BAPHandler state used to indicate ready to go state - not used.

stateBH_AdaptAndCoding_update 

BAPHandler state used to update the BAP properties after adaptation and/or codding change.

stateBH_AdaptAndCoding_PSO_update 

BAPHandler state used to update personalized BAP properties after adaptation and/or codding change - not used.

stateBH_FunctionList_update 

BAPHandler state used to update FctList after adaptation and/or codding change - not used.

stateBH_FactorySettings 

BAPHandler state used to reset to defaults BAP properties after receiving FSG_Control request.

stateBH_Stop 

BAPHandler state used to indicate end of work - not used.
