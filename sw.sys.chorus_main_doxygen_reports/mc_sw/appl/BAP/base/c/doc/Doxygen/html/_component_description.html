Component Description

General

The BAPHandler_PSO_ImExport library was designed to provide a complete set of functions which allow each BAPHandler to act as Active Participant and communicate with OCU.
It has to be used by any BAPHandler SWC explicitly by providing by specific BAPHandler SWC required set of input data (structures, buffers and pointers to specific functions)
to serve by BAPHandler_PSO_ImExport library the role of Active Participant in the name of and from the content of BAPHandler SWC.

The common library BAPHandler_UtilityLib collects some common defines, enums, types and functions which can be used by any SWC, however it was originally designed to be
used by BAPHandler modules.
