Detailed Description

The structure designed to store all mandatory elements (value and flags) used by single personalized parameter.
This structure is used to store information in NvM in the form usefull to be used by BAPHAndler_PSO_ImExport library.
NOTE:
This structure shall be used by all BAPHandler components defined for ICAS1 project to collect informations about each personalized parameter. It needs to be added also
in the SST file, otherwise, there will be problems with stublib generation.
