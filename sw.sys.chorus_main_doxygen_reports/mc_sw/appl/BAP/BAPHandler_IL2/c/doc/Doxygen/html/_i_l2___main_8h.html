Detailed Description

This file provides handling of BAP communication between actual Dimming
Master module and BAP protocol.

Source
IL2_Main.h
Revision
2.0
Author
Wojciech Sowinski (uib18192), Artur Redosz (uidp7980)
Date
2022-08-19

It hosts BAP InteriorLight2 (IL2) LSG.
It main purpose is to provide brightness value from user (via HMI) to
Dimming Master and store its personalised values accordingly to user slots.

This file can be included by other modules

◆ IL2_Cycle()

void IL2_Cycle ( void  )

Cyclic function of the InteriorlLight2 module. Called every 20ms.
It hosts InteriorlLight2 State Machine and calls specific functions dependingly on what state currently InteriorlLight2 is running.

The modules of functions are called cyclicly by BAPHandler_IL2_TaskCyclicEvent,
depending on the IL2_state state.
