Detailed Description

This file provides handling of BAP communication between actual Dimming
Master module and BAP protocol.

Source
IL2_Main.c
Revision
2.1
Author
Wojciech Sowinski (uib18192), Artur Redosz (uidp7980)
Date
2022-11-28

It hosts BAP InteriorLight2 (IL2) LSG.
It main purpose is to provide brightness value from user (via HMI) to
Dimming Master and store its personalised values accordingly to user slots.

This file can be included by other modules

Initializeation of BAP LSG functionality.

This function respond for initialize BAP LSG functionality sending InitSenBuffer requests with current BAP properities, enabling BAP
communication and postponing handling of BAP requests if needed.

Remarks

Input->Output-Correlation
If initialization was not done yet ( initialization_setup_state is FALSE) and BAP communication is disabled, initialize BAPHandler_IL2
