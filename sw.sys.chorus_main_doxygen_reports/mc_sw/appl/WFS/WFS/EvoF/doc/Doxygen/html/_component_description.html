Component Description

General

The purpose of the immobilizer(WFS) is to differentiate between authorized and unauthorized users and to either allow or not allow the operation of the vehicle.
The main task of the WFS master is to authenticate the vehicle key after various triggers are set off by the key coil or the Kessy system and to enable the requesting ECU if the key authentication is successful or to lock it if the key authentication is not successful.
