This function is called right after FFB_SKC learning process has succesfully ended.
It clears all obsolete FFB Keys and trigger server function responsible for learning (storing) data such as TransponderIDs or Rolling Code Counters (WCZ) in the ICAS1 NvM for all received FFB Keys.
