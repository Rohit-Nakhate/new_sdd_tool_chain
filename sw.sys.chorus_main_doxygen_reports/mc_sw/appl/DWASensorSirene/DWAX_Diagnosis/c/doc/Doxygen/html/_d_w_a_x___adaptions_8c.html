static boolean CheckDwaxDiagAvailable ( Dcm_OpStatusType  OpStatus,
Dcm_NegativeResponseCodeType *  ErrorCode 
)
static

CheckDwaxDiagAvailable function used to check the state of the car.

Function used to check in witch state the car currently is. This check refers to VWMEB-41937.
When the car is being externelly locked, diagnostics shall NOT be performed on listed in file header diagnostics request.
When diagnostics are rejected, NRC shall be set to DCM_E_GENERALREJECT.
