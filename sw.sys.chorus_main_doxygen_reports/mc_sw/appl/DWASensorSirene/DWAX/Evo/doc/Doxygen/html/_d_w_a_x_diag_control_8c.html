Detailed Description

DWAX diagnostic wrapper to check if diagnostics services are allowed.

Source
DWAXDiagControl.c
Revision
version
Author
uic20816
Date
Mar 30, 2022

This is to fulfill requirements that we shall reject diagnostics services when car is externally locked. All listed in file diagnostics request goes through addtional check CheckZvDiagAvailable. If car is currently in state externally locked there is NO diagnostics possible to perform and diagnostic shall return value DCM_E_NOT_OK with NRC DCM_E_GENERALREJECT

static boolean CheckZvDiagAvailable ( Dcm_OpStatusType  OpStatus,
Dcm_NegativeResponseCodeType *  ErrorCode

static

CheckZvDiagAvailable is use to check in witch state is currently car. This check refers to VWMEB-41937

When car is being externelly locked diagnostics shall NOT be performed on listed in file header diagnostics request.

When diagnostics are reject NRC shall be set to DCM_E_GENERALREJECT.

Returns
boolean - TRUE when diagnostics are possible, FALSE if NOT possible
