Verification Criteria

SW Unit testing is accomplished through SWATT. The test cases are based on this design, and also cover the configuration options. Expected test coverage (100%) and type of test coverage (C1). For more details please see Test List.

Verification of Resource Consumption

DWAX is an extension of DWA, which is a subcomponent of the ZV_Lib and the resource consumptions are estimated for the entire composition.
Esitmatied Resource consumption for component:

Component CPU Load ROM RAM NVM
DWAX 0.0, 0.12%, 0.0 8.3 KB 0.12 KB 0.006 KB

Note The CPU value is for each processor (Core0, Core1 and Core2).
