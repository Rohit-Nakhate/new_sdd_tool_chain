External Interfaces and Dependencies

External Interfaces

The external interfaces are represented by the generated RTE based on the DWAX_SstSr.xml sst file.
DWAX_SstSr.xml is exported from DWA_SST.xlsx with the mapping set in the DWAX_SstSr.xsd file.
