void DWAX_Adapter ( void  )

Adapter function of the DWAX module.

Function handles sending arming and Master code signals.

Precondition
N/A
Postcondition
N/A
Parameters
void
Returns
void
Globals
DWAX_Data.StateOfMasterCode [in]
Pim_DWAX_SireneMasterCode [in]

DWAFlagScheduleTableRequest [in]
DWAX_TASK_scharf [in]
DWAX_TASK_unscharf [in]
Input->Output-Correlation
If status of DWAX_Data.StateOfMasterCode is either DWAX_MASTERCODE_GENERATED (10U) or DWAX_MASTERCODE_SAFEFAIL (11U), DWA will trasmit to the Siren the MasterCode in order to arm the Siren.
Pim_DWAX_SireneMasterCode used to store the MasterCode.
DWAXMRRAIMUNDFEARS enables or disables parts of code.
DWAFlagScheduleTableRequest, DWAX_TASK_scharf, DWAX_TASK_unscharf used to check if the desired state of arming was reached.

void DWAX_InitVarFunc ( void  )

Initialize the variables for the DWAX module.

This function is called by ModelsTask(Scheduler) before starting the model (MmDoSetup is called). The code here must be short.

N/A

Returns

Input->Output-Correlation
DWAX_boRun is set to TRUE.
dwax_msgInitDelayCounter_u8 is initialised to 0.
CAN_TimeoutDelayElapsed_bo array elements are initialised to 0 for all 3 CANs (CCAN, EVCAN, KCAN).
DWAX_CAN_COUNTER used to store the number of CANS (3 CANs: CCAN, EVCAN, KCAN).
