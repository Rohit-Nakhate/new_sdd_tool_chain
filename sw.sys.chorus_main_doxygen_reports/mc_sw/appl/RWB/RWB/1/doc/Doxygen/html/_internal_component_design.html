Internal Component Design

Internal Component Design

The component is split into the follwing SW units:

  • RWB_Task.c - Code is called by scheduler which will init, start, run and stop the operation of RWB
  • RWB_Task.h - Header file for RWB_Task.c
  • RWB_Stubs.c - Code used to identify the SWC version and error codes of RWB
  • RWB_Stubs.h - Header file for RWB_Stub.c
  • RWB_ADAPTATION.c - Code is needed for handling write and reading requests of adaptions regarding RWB (generated)
  • RWB_ADAPTATION.h - Header file for RWB_ADAPTATION.c (generated)
  • RWB_MeasureValues_22.c - Auto-generated code for handling reading requests of measurement values regarding RWB (generated)
  • RWB_MeasureValues_22.h - Header file for RWB_MeasureValues_22.c (generated)
