Internal Component Design

Internal Component Design

The component KDBit_MQB consists of only the following SW unit: KdBit.c
The architectural overview of the component is described within the SWA at the following location in Enterprise Architect:
VW_ICAS1.Chorus.All_Cores.SoftwareDecomposition.ComplexDeviceDrivers.KDBit_MQB.SW Architecture

The SW unit contains an API ComputeKDBitInfo which is cyclically called by UdsTask under the API SystemInfo_Cyclic. This collects and computes the KD error bit flag. The signals SI_Sammel_SG_Fehler,SI_Sammelfehler, GW_KD_Fehler under Systeminfo_01 is updated based on KL15 status and GW failure present value. Setting or clearing of first bit of response of 0x2A28 is also done in this API.
The API KdBitNodemessageReceived notifies reception of node message when called.
