KL15 is equal to ON
AND
KL30 Voltage is in Operating Voltage Range "c" (UBmin = 9.0V and UBmax = 16.0V as defined in ยง4.7 [VW 80000])

Signal ESP_v_Signal is equal to 0
AND
Signal MO_QBit_Drehzahl_01 is equal to 0 (value: [gueltiger_Wert])
AND
Signal MO_Drehzahl_01 is equal to 0

Signal ESP_Rollenmodus_Deaktivieren is equal to 0 (value: [keine_Aenderung]) AND NOT in timeout.
