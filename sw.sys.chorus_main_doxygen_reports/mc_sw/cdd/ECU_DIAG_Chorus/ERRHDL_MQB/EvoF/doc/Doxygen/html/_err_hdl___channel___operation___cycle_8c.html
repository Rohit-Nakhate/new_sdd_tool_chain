◆ MFLNodeOperationCyclePreviousState

boolean MFLNodeOperationCyclePreviousState = 0
static

This array storages previous state of operation cycle for MFL node.

Bus operation cycle depends from bus state. When bus changes status from inavtive to active operation cycle starts, when bus change status from active to inactive operation cycle ends. TRUE means bus operation cycle is active, FALSE operation cycle is not active. MFL node can be coded on CAN or LIN bus. Coding on both busses is not possible.
