◆ OS_TASK_ErrHandlerTask()

void OS_TASK_ErrHandlerTask ( void  )

PUBLIC ROUTINES.

Task function Application Handler. This is a distribute task!!! Priority: 60 Schedule: FULL Type: EXTENDED

CAUTION:

Be aware, that the sequence of handling the events is not random! The first event to process is always the start event. The last event to process must be always the stop event. The priority in general is to process all error setting events an walk through events first (events updating the state) and then the events deleting some error contents.
