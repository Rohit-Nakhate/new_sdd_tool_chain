Component Description

General

The component ErrHdl_MQb (Error handler) deals with detection of error on CAN, LIN and VLAN networks and handles the corresponding DTC and enable conditions.

There are several different kinds of errors related to CAN self-diagnostics:

  • CAN BusOff Error
  • CAN Single-Wire Error
  • CAN No Communication i.e. no rx from any ECU on the bus
  • CAN message timeout error
  • CAN message _alt bit received error (from a Gateway, for some messages)
  • CAN E2E Errors
  • CAN ECU defect errors / coding errors / data errors (range of values)

There are several different kinds of errors related to LIN self-diagnostics:

  • LIN Bus - Electrical Failure
  • Slave - No communication
  • Slave - Implausible frame
  • Slave - Implausible Signal
  • Slave - Defective
  • Slave - Coding error
  • LIN Bus - no communication

There are errors related to VLAN self-diagnostics :

  • VLAN message timeout error of ECU

It also handles DTCs related to High temperature and Voltages.
