Detailed Description

HAL Wakeup Reason Type

This is the structure for any wakeup pin.
The element "wakeup_unit" is the WKPU register nummer (0 or 1).
The element "bit_mask" is the bit_mask in the WKPU register.

Remarks
A special value WAKEUP_UNIT_INVALID means what pin is not wakeable.
