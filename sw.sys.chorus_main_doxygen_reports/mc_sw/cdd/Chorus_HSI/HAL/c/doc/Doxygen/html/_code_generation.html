Code Generation - HAL_GEN

HAL_GEN general

The HAL_GEN tool uses the Port.xdm and Adc.xdm files to generate code.

The items to be generated need to be specified in hal_cfg.xml files.

HAL_GEN config

The hal_cfg.xml configuration file contains:

  • CAN mappings to ports from Port.xdm and ADC channels from Adc.xdm
  • LIN mappings to ports from Port.xdm
  • GPIO pins mappings to Port.xdm
  • Wakeup reasons mappings to Port.xdm
  • mapping of ADC channels from ADC.xdm

There can be multiple mappins for the same item.
All the map elements will be used to search in the *.xdm files.
The first map will be used a name for generated files.
The CAN and LIN channels need to have the same mapping as in the enum TComChannel from ConfigChannels.h.

HAL_GEN code generation

Use the HAL_GEN CMake target to generate the code.
