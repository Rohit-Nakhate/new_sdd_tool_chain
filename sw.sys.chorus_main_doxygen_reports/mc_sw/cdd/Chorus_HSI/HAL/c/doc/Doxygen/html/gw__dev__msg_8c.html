Send GW developer messages GW_DEV_01, GW_DEV_02, GW_DEV_ETH on Diagnose CAN.

This function send GW developer messages GW_DEV_01, GW_DEV_02, GW_DEV_ETH on Diagnose CAN by filling two 12bit and performing the generic direct trasmission with control options.
