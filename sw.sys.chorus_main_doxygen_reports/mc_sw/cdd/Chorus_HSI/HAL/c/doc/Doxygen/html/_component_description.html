Component Description

General

The HAL component provides interfaces for access to different peripherals of the Chorus ECU such as GPIO, ADC, HW timers and Wakeup Unit.

The configuration items are genererated based on the Port and ADC configuration for the AUTOSAR modules.

Use CMake HAL_GEN target to generate the HAL configuration. See Code generation.
