◆ HAL_SecondaryCores_Disable()

void HAL_SecondaryCores_Disable ( void  )

Helper function to disable secondary cores.

Mark core1, core0 and HSM as inactive for the next Mcu mode transition.
Before the transition to Standby is done, all cores except Core 2 are to be stopped.
If this is not done sporadically on a wakeup from Standby Core1 or Core 0 are already starting while Core 2 is in initialization of the RAMs. Here obviously the DMA is disturbed and doesn't do it's work. The complete system doesn't startup properly.
