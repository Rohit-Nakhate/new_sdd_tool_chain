This module reads current HW version and base variant from defined OTP flash area (set in .pdr/.vpdr). APIs are available for other SW modules to request the HW version and base variant in a readable format.

void HardwareVersion_GetHardwareVersion ( HardwareVersionType *  hwVersion)

Function that selects and stores the corresponding hardware version based on hardware version information.

Based on the OTP data (hardware version information) read from OTP flash memory, this function creates an 32 bit\ integer by combining HW version IDs and based on this integer, it will select the corresponding hardware version type.

Function that reads the hardware version information from the OTP (one time programmable memory) flash memory.

If the hardware version wasn't read before, all the necessary informations will be read and stored into a struct.
