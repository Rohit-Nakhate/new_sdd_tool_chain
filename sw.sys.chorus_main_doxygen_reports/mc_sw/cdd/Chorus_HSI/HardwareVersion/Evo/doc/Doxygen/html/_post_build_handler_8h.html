void PostBuildHandler_McalInit ( void  )

Post build configuration function.

According to the HW version type, on MCAL (Microcontroller Abstraction Layer) level the Port Driver, ADC Driver, ICU Driver modules are initialized.
Then, on HAL (Hardware Abstraction Layer) level the KL30 voltage measurment is enabled, the HAL configuration is set, channel configuration is prepared and the Unit is setup and wakeuped before sleep.
This function will be called in the context of EcuM module and the call is generated by the Configuration tool.
NOTE: This design was made in this way to take into consideration if there will be other configurations for specific HW versions.

PORT_PIN_LEVEL_HIGH will be used to set the KL30 pin to high.

void PostBuildHandler_McalInit_LIN4_DWA_mode ( void  )

Post build configuration function specific for DWA and LIN pooling mode.

According to the HW version type, on MCAL (Microcontroller Abstraction Layer) level the Port Driver module
is initialized. Then, on HAL (Hardware Abstraction Layer) level the HAL configuration is set, channel configuration is prepared and the Unit is setup and wakeuped before sleep.
Only LIN4 pins are configured.
This function is made for a specific case of configuration. It will be called in the context of LowPower_Main module and it represents the case of cyclic wakeup from low power mode.
NOTE: This design was made in this way to take into consideration if there will be other configurations for specific HW versions.
