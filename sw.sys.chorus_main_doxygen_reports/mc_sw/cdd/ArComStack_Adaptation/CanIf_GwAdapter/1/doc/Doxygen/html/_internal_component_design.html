Internal Component Design

The component is split in two units: CanIf_GwAdapter's static code and configuration. The configuration part is generated based on an export from tresos.
The relevant part are the Cand and CanIf configuration.

Dynamic Behavior

The Module interractions can be split in three parts: transmission, reception and dynamic channel config.
Transmission.

The upper layer (CanTp) is calling the transmission function with parameters. It expects by successful transmission to receive a confirmation.

Dynamic channel configuration.
The dynamic part is designed for interractions with DoIP. It provides mechanism for Tx Confirmation and Rx Indication.
This part is not used in ICAS1/ICASevo but it is still maintained for other projects
