Detailed Description

This module implements the LIN ISR functions.

There are 3 types of interrupts for each LIN interface: Lin[x]RxIsr - interrupt for reception Lin[x]ErrIsr - interrupt for error frame Lin[x]TxIsr - interrupt for transmit Where x represents the interface identifier: starts from 0 and goes till LIN_IFC_NR_MAX
