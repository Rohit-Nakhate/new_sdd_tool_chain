Detailed Description

This module provides the implementation of the main task loop of the LinTask.

The module includes also necessary functions for the the standard interface definition and the task handling by the module manager. The task function definition is called directly in the scope of the OSEK operating systems.

◆ LIN_EV_MASK

#define LIN_EV_MASK
Value:
(LinStartEv + \
LinStopEv + \
LinBaseTickEv)

This is the the mask for the LIN events waited for.

The defined mask will be given as parameter for the WaitEvent() API call in the LinTask.
This API call puts the calling task into the waiting state until one of the specified
events is set. If one or more is already set, the task remains in the running state.

Remarks
For LIN task there are three events to be waited for: LinStartEv, LinStopEv and LinBaseTickEv.

◆ LinBaseAlarm_active

boolean LinBaseAlarm_active
static

Flag for the activation of cyclic alarm of LinTask

The flag will be updated when the cyclic alarm of the LinTask is started in
LinStartTimer() function and stopped in the LinStopTimer() function.
We need to keep track of the status of the alarm and do not cancel the alarm if it is not started.

Warning
Canceling an alarm not previously started will result in setting an OS error
