Component Description

General

The LIN Integration component represents an upper layer for the LIN_Com drivers.
The component uses the LIN FlexDriver LLD from EB and LIN Config which represents the generated configuration and signal interface that is LDF dependent.
Will provide in turn the necessary LIN interfaces for other modules as described in SWA.

This component is split in 4 other subcomponents:

  • LIN task: provides the implementation of the main task loop of the OSEK LinTask.
  • LIN configuration: Maps the automatically generated configuration for LIN with the actual implementation in LIN Integration component.
  • LIN ISRs: Implementation of LIN ISR functions.
  • Scheduling of LIN tables: Responsible for switching the LIN schedule tables when the requests for switching was received from other components.
