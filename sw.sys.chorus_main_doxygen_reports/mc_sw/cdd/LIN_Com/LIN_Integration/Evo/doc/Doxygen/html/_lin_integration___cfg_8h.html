const LinScheduleConstDataT LinScheduleConstData[LIN_IFC_NR_MAX]

Constant array that contains the schedule tables for all the LIN interfaces

This will be used by other Lin Integration subcomponents to manage the scheduling of the LIN tables. The initialization of the data structures is in accordance with the information given in the LDF files.

Remarks
The size of the array corresponds to the number of LIN interfaces and is defined by LIN_IFC_NR_MAX
