The OperationCycleMenagement module.

Revision

DateTime

The function of OperationCycleMenagement module is to recognize & validate Driving Cycle(DCY) and Warm Up Cycle(WUC). To decrement Aging Counter for DTCs and to increment aging counter for CAN message Diagnose_01. The OCM module will monitor the OBD_DrivingCycle until first Set & returns/writes the OBD_DrivingCycle State which is stored in NvM.
