Detailed Description

The OperationCycleMenagement module.

Source
OperationCycleMenagement.c
Revision
#5
Author
uie90984
Date
2022/04/06

The function of OperationCycleMenagement module is to recognize & validate Driving Cycle(DCY) and Warm Up Cycle(WUC). OperationCycleMenagement module decrement Aging Counter for DTCs and to increment aging counter for CAN message Diagnose_01. and it will monitor the OBD_DrivingCycle until first Set & returns/writes the OBD_DrivingCycle State which is stored in NvM.

Verification criteria
SW Unit testing is accomplished through SWATT. The test case are based on this design, and shall also cover the configuration options.
