Component Description

General

The function of OperationCycleMenagement module is to recognize & validate Driving Cycle(DCY) and Warm Up Cycle(WUC).
OperationCycleMenagement module decrement the Aging Counter for DTCs and to increment aging counter for CAN message Diagnose_01.
and it will monitor the OBD_DrivingCycle and it returns/writes the OBD_DrivingCycle State to NvM.
