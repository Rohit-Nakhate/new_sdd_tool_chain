Internal Component Design

The component DiagnosticTimers consists of only the follwing SW unit: DiagnosticTimers
The architectural overview of the component is described within the SWA at the following location in Enterprise Architect:
VW_ICAS1.Chorus.All_Cores.SoftwareDecomposition.ComplexDeviceDrivers.UDS_Service_Chorus.SW Architecture.DiagnosticTimers.SW Architecture

DiagnosticTimers module deals with the Timers for maintaining diagnostic activity. Hence it was decided to keep this module under High power mode under Core2 DiagTask.
