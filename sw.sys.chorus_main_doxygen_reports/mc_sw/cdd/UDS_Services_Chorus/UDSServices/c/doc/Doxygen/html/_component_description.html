Component Description

General

Unified Diagnostic Services (UDS) is an automotive protocol that lets the diagnostic systems communicate with the ECUs to diagnose faults and reprogram the ECUs accordingly (if required).
It is called unified because it combines and consolidates all the standards like KWP 2000, ISO 15765 and ISO 14229 others.
The function of UDS software stack is to store the fault code in the ECU memory for every issue that occurs in the vehicle and transfer it (to the client side) as and when required.
