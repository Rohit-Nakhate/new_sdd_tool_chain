static void processOBDLocking ( boolean  deactivationRequestedBy0x0300)
static

This function is used to check the status of ORU and based on the state, diagnosis routing state is changed.

  • This function is used to activate the locking of diagnosis routing or to deactivate the locking of diagnosis routing. Activation is dependent on ORU_status (Online Remote Update). Deactivation is dependent on ORU_status, on deactivation request by routine 0x0300 and on clamp 15 state.
