Detailed Description

This module deals with the Task API and it also deals with the 'Task Setup', 'Task Stop', 'Task Alive'.

Source
UdsTask.c
Revision
version
Author
uie90984
Date
2022-04-06


This module provides the implementations for the task frame of the complete UDS-stack and deals with cyclic functions for HSM, UDS, SFD, SecDiag modules.
It deals with cyclic tasks for diagnostic preconditions.

Verification criteria
SW Unit testing is accomplished through SWATT. The test case are based on this design, and shall also cover the configuration options.
